# 🐞 Bug Report

## Summary

The **Preview** page incorrectly displays teaching elements before any teaching elements have been added to the course structure.

> Example: Opening the Preview immediately after creating a course structure shows teaching elements even though none have been added.

## Preconditions

- User is logged into the LMS application.
- A course named **Defect Avenger** exists.
- No teaching elements have been added to the course structure.

---

## Steps to Reproduce

1. Log in to the LMS application with valid credentials.
2. Navigate to **Course Management**.
3. Search for the course **Defect Avenger**.
4. Click **Add Course Structure**.
5. Without adding any teaching elements, click **Preview**.
6. Observe the preview page.

---

## Expected Result

The Preview page should not display any teaching elements when none have been added to the course structure.

---

## Actual Result

The Preview page displays teaching elements even though no teaching elements have been added.
