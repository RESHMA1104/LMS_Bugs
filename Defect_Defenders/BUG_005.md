# 🐞 Bug Report

## Summary

The **Course ID** takes an unusually long time to generate while creating a new course. During this delay, the field remains in the **"Generating course ID..."** state.

> Example: After entering the required course details, the Course ID does not generate within a reasonable time.

## Preconditions

- User is logged into the LMS application.
- User has permission to create courses.
- User is on the **Create New Course Setup** page.

---

## Steps to Reproduce

1. Log in to the LMS application with valid credentials.
2. Navigate to **Course Management**.
3. Click **Add Course**.
4. Fill in the required fields:
   - Course Client
   - Service Type
   - Service Model
   - Course Category
   - Course Name
5. Observe the **Course ID** field.

---

## Expected Result

The **Course ID** should be generated automatically within a few seconds after entering the required details.

---

## Actual Result

The **Course ID** field remains in the **"Generating course ID..."** state for an extended period before displaying the generated Course ID.
