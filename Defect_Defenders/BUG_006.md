# 🐞 Bug Report

## Summary

The **Course Management** page fails to load the course table after navigation. Instead of displaying the available course records, the application shows **"No users found"**, preventing users from viewing existing courses.

> Example: Navigating to the Course Management page displays an empty table with a "No users found" message despite existing course records.
