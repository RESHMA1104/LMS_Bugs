# 🐞 Bug Report

## Summary

The **Service Model** dropdown displays options that are not relevant to the selected **Service Type** while creating a new course.

> Example: Selecting **Front End Development** as the Service Type shows unrelated Service Model options.

## Preconditions

- User is logged into the LMS application.
- User has permission to access **Course Management**.
- User is on the **Create New Course Setup** page.

---

## Steps to Reproduce

1. Log in to the LMS application with valid credentials.
2. Navigate to **Course Management**.
3. Click **Add Course**.
4. Select a **Course Client** (e.g., *Jamocha*).
5. Select a **Service Type** (e.g., *Front End Development*).
6. Open the **Service Model** dropdown.
7. Observe the available options.

---

## Expected Result

The **Service Model** dropdown should display only the models mapped to the selected **Service Type**.

---

## Actual Result

The **Service Model** dropdown displays options that are unrelated to the selected **Service Type**.
